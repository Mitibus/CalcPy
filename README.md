
# CalcPy

A simple Python calculator made with Tkinter


## Installation

Install my-project with python

1. First clone the project
```bash
  git clone https://github.com/Mitibus/CalcPy.git
```
2. Install all the dependencies
```bash
  pip install tkinter
```
3. Run the program
```bash
python main.py
```
